# Uniform Shop ERP

## Overview

A Laravel-based ERP and Point of Sale (POS) system for managing school uniform production, finished inventory, sales, customers, and reporting.

---

## Technologies

- Laravel 12
- PHP 8.2
- MySQL
- Blade
- Vite
- Git
- GitHub

---

## Planned Modules

- Authentication
- Role Management
- Categories
- Products
- Product Variants
- Stock Entries
- Inventory Management
- Sales (POS)
- Customers
- Reports
- Dashboard
